
1. Cara Log in Github di VScode
   - Tekan Ctrl + shift + P
   - isi GitHub: Sign in
   - Lalu log in sesuai akun
   - Konfigurasikan 
    git config --global user.name "NamaGithubKamu"
    git config --global user.email "EmailGithubKamu"


2. Cara Menarik File dari Github ke VScode 
   - Ketik git clone "Link repo githubnya"
   - Ubah alamat eksekusinya ke file yang diambil dengan cara cd ./fileyangdiambildariGithub
   - Nanti ada tulisan biru diujung kayak master atau kita namanya itc

3. Cara Up to date dengan file yang ada di github 
   - ketik git pull biar up to date dengan github, nanti filenya bakal sama kayak yang di github. 

4. Cara Mengupload File dari VScode ke Github
    - Ketik git add , ini buat milih mana aja file yang mau disave. Cuman lebih sering semua file 
      itu disave, jadi ketiknya git add .
    - Ketik git commit -m "Pesan buat apa yang diubah"
    - git push buat ngesave di repo github
